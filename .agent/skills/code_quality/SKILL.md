# Code Quality and Visual Excellence Skill

Every code change must adhere to the highest standards of quality, both functionally and visually.

## Instructions

1. **Prioritize Visual Excellence**: Always implement designs that "WOW" the user. Use curated, harmonious color palettes, modern typography, smooth gradients, and subtle micro-animations. Avoid generic colors.
2. **Semantic HTML**: Always use appropriate HTML5 semantic elements and ensure all interactive elements have unique, descriptive IDs for browser testing.
3. **No Unused Code**: Clean up any unused imports, variables, or commented-out old logic when refactoring.
4. **Follow Best Practices**:
    - For HTML/JS: Vanilla CSS is preferred over Tailwind unless specified. 
    - For Python: Follow PEP-8.
    - SEO: Include proper title tags, meta descriptions, and heading structure (`<h1>` per page).
5. **Robust Error Handling**: Ensure functions catch Edge cases and avoid silent failures. Review the browser or runtime console output after major changes.
